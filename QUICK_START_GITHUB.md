# GitHub quick start

1. Create a blank repository named `royal-road-longitudinal-lab`.
2. Upload this project or open it in Codex with GitHub access and push it.
3. In **Settings → Secrets and variables → Actions**, add `RR_USER_AGENT`.
4. In **Settings → Actions → General**, set workflow permissions to **Read and write**.
5. Open **Actions → Royal Road longitudinal snapshot → Run workflow**.
6. Inspect `reports/validation_latest.json`. A complete RS run has 50 rows for all six `rs_*` sources.
7. Download `royal-road-longitudinal-latest` from the workflow artifacts or use `data/export/royal_road_longitudinal_latest.zip`.
